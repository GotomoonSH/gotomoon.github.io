## 线性回归输出是一个连续值，因此适用于回归问题。

### 线性回归的基本要素

矢量计算表达式
``` bash
from mxnet import nd
from time import time
a = nd.ones(shape=1000)
b = nd.ones(shape=1000)
```
向量相加的一种方法是将两个向量按照元素逐一做标量加法：
``` bash
start = time()
c = nd.zeros(shape=1000)
for i in range(1000):
	c[i] = a[i] + b[i]
time() - start
```
