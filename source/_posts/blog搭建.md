## 简单介绍
* Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章。
	> [Hexo官方文档](https://hexo.io/zh-cn/docs/)
* Node.js 是一个让 JavaScript 运行在服务端的开发平台。
* github 代码托管平台。（备份本地blog文件）


1. 安装Node.js

2. install Hexo
	> 什么是Hexo
	熟悉基本Hexo命令
	hexo clean
	hexo g
	hexo s
	hexo d


3. 建立git连接
生成ssh密钥，



4. 备份hexo文件

5. 发布文章
